Nice alternative to mocks, generate stub implementation

Here is nice go library named ‘moq’ that helps you to easily test dependent code but without verbose mocks setup. The idea is to generate trivial implementations and just save all the calls as a list preserving all the arguments. Using this you can simply assert all the invariants and stop to write long mock setup.
